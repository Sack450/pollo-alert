# pollo-alert 🤙 🐔

Un sencillo proyecto web que implementa una alerta modal interactiva que se muestra al cargar la página. El modal incluye una imagen (`si.png`) y puede ser cerrado mediante un botón o presionando la tecla **Escape**.

## Características

- **Diseño limpio:** Estilos integrados en CSS responsivo.
- **Interactividad básica:** Cierre de modal con el botón "Cerrar" o mediante el teclado (tecla Escape).
- **Estructura semántica:** Uso de roles de accesibilidad (`role="dialog"`, `aria-modal="true"`, `aria-labelledby="alertTitle"`).

## Archivos del Proyecto

- `pollo_alert.html`: Archivo principal con la estructura HTML, estilos CSS y comportamiento JavaScript.
- `si.png`: Imagen que se muestra dentro de la alerta modal.
